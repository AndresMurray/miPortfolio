Proyectos

Usé Python para simplificar procesos repetitivos, incluyendo la gestión de archivos.

Creé consultas SQL para procesar datos y generar reportes.

Creé scripts en Bash para automatizar tareas administrativas, como la limpieza de logs, la gestión de permisos de archivos y la generación de reportes del sistema.

Modele y normalicé bases de datos, asegurando consistencia e integridad referencial.

Diseñé e implementé soluciones orientadas a objetos. Además, realicé pruebas automatizadas con JUnit.

Desarrollé una aplicación de escritorio utilizando .NET con Programación Orientada a Objetos (POO) para gestionar datos en una base de datos SQL. La aplicación permite realizar operaciones como agregar, editar y eliminar registros, con una interfaz gráfica amigable y una conexión eficiente a la base de datos.

Tecnologías utilizadas: C#, .NET Framework, SQL Server, POO

Objetivos:

  • Implementar una estructura robusta y reutilizable utilizando POO.
  • Desarrollar una conexión a base de datos SQL para almacenar y recuperar información.
  • Crear una interfaz gráfica que permita una experiencia de usuario intuitiva.
